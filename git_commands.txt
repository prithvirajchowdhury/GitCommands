1)To clone a repository : git clone

2)To veiw information about remote repo : git remote -v 

3)To view all branches of the repository : git branch -a

4)To initialize a git repo : git init

5)To add files to the staging area: git add <filename>

6)To remove files from staging: git reset <filename>

7)To se what has changed in working directory : git diff

8)To create branch: git branch <branch_name>

9)To remove any staged file from the working directory: git rm -f <file_name>

10)To switch to a branch : git checkout <branch_name>

11)To pull others changes: git pull origin <branch_name> 

12)To see merged branch : git branch --merged

13)To merge a branch with master:git merge <branch_name>

14)to push you changes: git push origin <branch_name>

15)To undo changes : git checkout -- <fileName> for all files git checkout -- .

16)To revert commited changes: git revert <commit_id> | git revert -n <commit_id>

17)To reset to a particular commit id: git reset --hard <commit_id>
